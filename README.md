# Inditex

_ API para Inditex

### Instalación 🔧

_ Tener instalado JDK 17
- Tener instalado Maven

## Despliegue 📦

_ spring-boot:run -f pom.xml
- Para ver el API: http://localhost:8080/swagger-ui/index.html#/
